# 清能笔记 · 能源环保通识课

能源环保科普网页，纯前端单文件实现（HTML + CSS + JS，无需构建、无第三方依赖），适合作为科普宣传页或课堂互动素材使用。

**在线访问：[https://tracywang0904.github.io/energy-eco/](https://tracywang0904.github.io/energy-eco/)**

## 内容板块

1. **基础概念** — 9 个可点击的概念卡片：可再生能源、化石能源、温室效应、碳达峰碳中和、节能减排、碳足迹、能源效率、循环经济、国际气候合作。点击卡片弹出详情弹窗，含 400-500 字详细介绍、听读按钮 + 进度条（朗读时逐句高亮，点击文中任意句子可跳转到该句），以及权威信息来源链接
2. **为什么要做** — 6 条科普理由（气候变化、资源枯竭、健康、生态、经济、国际共同行动），参考数据引用 IPCC 报告原文链接
3. **算一算：个人碳足迹计算器** — 选通勤方式 / 每周里程 / 饮食结构 / 冰箱数量 / 空调每日使用时长，实时算出年度碳排放（kg CO₂e），并换算成"相当于种几棵树才能吸收""相当于烧多少升汽油"两种直观类比，附通勤/饮食/家电三项占比明细
4. **生活实践** — 居家节能 / 低碳出行 / 减少浪费 / 节约用水四类共 20 条可打卡的习惯清单，部分标签附官方文件链接
5. **小游戏「变废为能」** — 12 种常见垃圾（每类 3 件）拖拽到正确分类桶，再选择对应的能源转化方式，提交后给出对错讲解
6. **随堂测试** — 31 题题库随机抽取 10 题、选项随机打乱，答题结束根据得分给出鼓励语和逐题讲解

页面中多处附有官方科普资料链接，包括国家能源局、IPCC、国务院、住建部、发改委、联合国等公开信息来源。

## 技术说明

- 单个 `index.html` 文件，字体（Baloo 2 / Nunito）以 base64 内联，离线也可直接双击打开
- 无构建工具、无外部依赖，可直接部署到任意静态托管平台
- 支持浅色 / 深色两套配色，跟随系统或站点主题自动切换

## 碳足迹计算器的系数

计算器里的排放系数都是为了教学直观、经过简化的估算值，不是精确核算，写在 `index.html` 的 `carbon footprint calculator` 脚本开头：

- 通勤：步行/骑行 0、公交地铁 0.06、电动车 0.10、燃油车 0.19（kg CO₂/公里），乘以每周里程再乘以 52 周
- 饮食：素食为主 600、均衡饮食 1100、肉食为主 1800（kg CO₂/年，粗略对应不同肉类摄入水平的年度饮食碳强度）
- 家电：每台冰箱 220 kg CO₂/年；空调按"每天使用小时数 × 108 kg CO₂/年"估算（对应约 1.5kW 机型、120 天制冷季节）
- 类比换算：1 棵树年吸收 ≈18kg CO₂，1 升汽油燃烧 ≈2.3kg CO₂

如果要换成更贴近实际的系数，直接改这几个常量即可，不用动其他逻辑。

## 听读音频

9 个概念全部已接入 CosyVoice 生成的真实朗读（`audio/*.mp3`）。如果文件缺失或加载失败，会自动改用浏览器内置的语音合成（Web Speech API）朗读作为兜底，不影响功能完整性。

```
audio/renewable.mp3       可再生能源
audio/fossil.mp3          不可再生能源
audio/greenhouse.mp3      碳排放与温室效应
audio/carbon_peak.mp3     碳达峰 / 碳中和
audio/energy_saving.mp3   节能减排
audio/footprint.mp3       碳足迹
audio/efficiency.mp3      能源效率 / 能效标识
audio/circular.mp3        循环经济
audio/cooperation.mp3     国际气候合作
```

**替换某个音频**：直接用同名文件覆盖 `audio/` 下对应的 mp3 即可，无需改代码。句子高亮默认按每句文字长度占全文（或所在分段）的比例估算起止时间；如果新音频的分段方式变了，需要同步更新 `index.html` 的 `CONCEPTS` 数据里对应概念的 `timestamps` 字段（`[{start:0,end:3.2}, ...]`，单位秒，顺序需与正文句子一致——句子拆分规则是按 `。！？；` 断句）。9 个概念的 `timestamps` 都是按“真实分段时长 + 段内按字数插值”的方法算出来的，可以直接参考现成的写法；`circular`（循环经济）那段还演示了如何在两个分段之间手动插入静音间隔。

## 本地预览

直接用浏览器打开 `index.html` 即可，或起一个本地静态服务器：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。
