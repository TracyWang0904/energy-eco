# 清能笔记 · 能源环保通识课

一个清新蓝绿色调的能源环保科普网页，纯前端单文件实现（HTML + CSS + JS，无需构建、无第三方依赖），适合作为科普宣传页或课堂互动素材使用。

**在线访问：[https://tracywang0904.github.io/energy-eco/](https://tracywang0904.github.io/energy-eco/)**

## 内容板块

1. **基础概念** — 可再生能源、化石能源、温室效应、碳达峰碳中和、节能减排、碳足迹
2. **为什么要做** — 气候变化、资源枯竭、健康、生态、经济等角度的科普理由
3. **生活实践** — 居家节能 / 低碳出行 / 减少浪费 / 节约用水四类可打卡的习惯清单
4. **小游戏「变废为能」** — 拖拽垃圾到正确分类桶，再选择对应的能源转化方式，提交后给出对错讲解
5. **随堂测试** — 16 题题库随机抽取 10 题、选项随机打乱，答题结束根据得分给出鼓励语和逐题讲解

## 技术说明

- 单个 `index.html` 文件，字体（Baloo 2 / Nunito）以 base64 内联，离线也可直接双击打开
- 无构建工具、无外部依赖，可直接部署到任意静态托管平台
- 支持浅色 / 深色两套配色，跟随系统或站点主题自动切换

## 本地预览

直接用浏览器打开 `index.html` 即可，或起一个本地静态服务器：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。
